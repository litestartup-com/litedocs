# LiteDocs

LiteDocs 是一个轻量级、基于 Markdown 文件夹的文档工具。编写 Markdown 文件，按文件夹组织，即可获得一个美观的文档网站。

## 为什么选择 LiteDocs？

- **零构建步骤** — 无需 npm，无需打包工具，纯 Python
- **约定优于配置** — 目录结构即配置
- **开箱即用** — 现代 UI，暗色模式，响应式布局
- **多语言** — 通过目录结构内置国际化支持
- **多文档** — 同时运行多个文档目录
- **可主题化** — 可切换的主题包

## 快速开始

```bash
pip install litedocs
litedocs serve ./my-docs
```

访问 `http://localhost:8000` 即可查看文档。
