# 开发文档

本节介绍 LiteDocs 的内部架构和贡献指南。

## 技术栈

- **Python 3.11+** — 运行时
- **FastAPI** — Web 框架
- **Jinja2** — 模板引擎
- **markdown-it-py** — Markdown 渲染
- **HTMX** — 局部页面更新
- **Tailwind CSS (CDN)** — 样式
- **Vanilla JS** — 无前端框架
